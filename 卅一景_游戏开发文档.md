# 项目概述

《卅一景》是一款 **文人悲情 / 双世界解谜 / 园林视角探秘** 互动叙事游戏。

玩家扮演美术学院古画修复专业研究生 **沈念**，在导师 **周鹤年** 的指导下，对文徵明《拙政园三十一景图》体系中的最后一页进行数字化修复。玩家将在 **现实修复工作室** 与 **画中拙政园** 之间穿梭，通过扫描观察、倒影视角、题诗比对、物件组合、草图显现与终章视角复原，逐步发现：

**第三十一景的问题，不是谁的笔法，而是谁的眼睛。**

第三十一景的画面本身一直存在，且笔墨成熟稳定。真正被遮蔽的不是一幅失踪画作，而是说明这个低位视角来源的边注、题签、辅助线和残字。王蘅曾发现这个视角，文徵明以自己的笔保存了她的眼睛；后人没有重画此景，而是在重装、配边、归档过程中将来源痕迹压入作品边缘。

MVP 目标：做出一段 **8\-12 分钟可完整演示** 的主线体验，稳定跑通五章核心流程，并完成三个文本结局之一。

当前前端实现状态：已完成 `landing`、`menu`、`prologue` 三个注册场景，以及核心引擎、存档、提示、物件匣、DeepSeek AI 服务、周鹤年对话面板和 AI 笔记本面板。第一章、第二章、第三章、终章和三结局仍属于后续 MVP 实现范围。

当前资源约定：正式图片统一存放在 `Web/public/images/`，代码中使用 `/images/文件名` 引用；`TestPic/` 只作为过程图片、生成图和临时参考图目录。

# 核心体验与设计目标

## **核心悬念**

- 第三十一景为什么采用了一个异常低、异常偏的观看位置？

- 页边装裱层下的“所见”、低位辅助线与“蘅”字残痕来自谁？

- “画非一人”究竟是指另一个画家，还是另一个观看者？

- 当证据高度可信但无法完全证明时，沈念应如何写入修复报告？

## **核心玩法**

- 扫描观察：放大镜、纸质分析、侧光照射。

- 双世界探索：现实修复工作室与画中拙政园切换。

- 视角谜题：水面倒影、镜像翻转、低位观看。

- 文本比对：题诗异文点击，组合“画非一人”。

- 物件组合：残砚显线，断簪揭灰。

- 终章复原：四问确认“谁在看 / 从哪里看 / 看到了什么 / 为什么来源被遮蔽”。

## **核心情感**

玩家不是在找鬼怪，也不是在证明“王蘅是另一位文徵明”。玩家要复原的是一个没有正式署名位置的人曾经留下的观看方式。

王蘅的动机不是“要被正名”，而是：

> 不必有名，不必有形。只要有痕迹。
>
>

最终体验应让玩家意识到：修复不仅是还原物质表面，也是在决定什么被继续隐藏、什么被重新看见。

## **设计目标**

- 让玩家体验“古画修复”的专业感：扫描、侧光、装裱层、异文、修复报告。

- 让玩家体验“进入画中园林”的志怪感：墨迹流动、倒影藏物、朱砂呼应、墙面草图。

- 让玩家体验“见证被遮蔽视角”的情感回收：从误以为王蘅是女画者，到理解她是空间观看者。

- MVP 不做开放世界，优先保证五章主线清晰、谜题目标明确、结尾情感完整。

# 不能违背的剧情基准

1. **画心即低视角成稿**

第三十一景的画面保留了王蘅发现的低位视角。文徵明完成画心时保留此视角，但没有另写来源说明。

2. **来源痕迹由王蘅留下**

“蘅”“所见”、低位辅助线、题记残字、草图与信件等来源痕迹均归王蘅本人留下。文徵明是消极保存者，不是主动留线人。

3. **后人没有重画第三十一景**

后人遮蔽的是说明来源的边注、题签、辅助线和残字，不是把第三十一景重画成另一幅画。

4. **王蘅不是可考历史人物**

王蘅为虚构人物。她代表曾参与文化创造、空间构思与审美判断，却没有被正式记录的观看者。

5. **王蘅不是被埋没的天才女画家**

她的草图比例不稳、线条笨拙，价值不在画技，而在发现了一个独特的园林观看位置。

6. **见证框架为单向**

王蘅的回声不指认当下的“你”，只是等了五百年的确认终于落地。

# 世界观与主要人物（王子轩负责）

序章结束后玩家“跌入”画中世界。之后每章结束自动返回现实世界，通过周鹤年对话、修复笔记本记录和“继续修复”按钮进入下一章。

现实世界存在轻量压力：**修复报告即将定稿**。若证据不足，第三十一景会被按“现存表层图像无异常”归档。该压力只通过文本呈现，不做硬倒计时。

# 五章流程总览（王子轩负责）

## 序章 · 残页

**主场景**：现实修复工作室。

**章节目标**：建立修复任务、双世界规则与核心悬念。

**核心异常**：

- 第三十一景表层图像稳定，并非“画风不像文徵明”。

- 装裱接缝与其他册页不同，似乎经历过二次揭裱。

- 背纸年代与其他页面略有偏差。

- 侧光下，页边装裱层下方隐约压着旧字残痕“……所见”与极淡低位辅助线。

- 表层取景位置很低，能同时容纳远香堂倒影、小飞虹弧线、梧竹幽居竹影。

**核心谜题**：扫描比对。

**玩家操作**：

1. 用放大镜检查装裱接缝与边缘题签压痕。

2. 用纸质分析查看背纸年代与局部修补区域。

3. 用侧光照射显出“所见”残字与低位辅助线。

4. 在侧光模式下点击旧字残痕与低位辅助线交会处。

**获得物件**：修复笔记本。

**章末结果**：墨迹流动，玩家跌入画中世界。

## 第一章 · 东园 · 兰雪堂至芙蓉榭

**主场景**：兰雪堂、缀云峰、芙蓉榭。

**章节目标**：让玩家理解王蘅以“倒影、低处、边缘、背面”的方式存在。

**关键场景**：

- 兰雪堂：匾额“兰”字有多余细笔，暂时只记录为异常。

- 缀云峰：低处石缝能看到远处水面，提示“有些景，只从低处出现”。

- 芙蓉榭：真实栏杆空无一物，水面倒影中挂着断簪。

**核心谜题**：水面倒影。

**玩家操作**：

1. 点击倒影中的断簪。

2. 阅读“倒影中的东西，要用倒过来的方式去拿”反馈。

3. 点击“镜像视角 / 翻转”按钮。

4. 倒影变正像后拾取断簪。

**获得物件**：断簪。

**关键线索**：断簪背面刻有极小“蘅”字。

**章末结果**：返回现实后，周鹤年解释“蘅，杜衡，是一种香草，也可比喻品性高洁的女子”，但提醒“一个字不能证明一个人”。

---

## 第二章 · 中园 · 远香堂至小飞虹

**主场景**：远香堂、小飞虹。

**章节目标**：推进玩家误读，让玩家以为王蘅可能是另一位未署名画者，同时引入“规范抹除”的声音。

**关键场景**：

- 远香堂：题诗墙与参考文献版本存在异文。

- 画案/小飞虹低处：发现残砚，砚池残留朱砂，砚背刻词。

- 旧批注残片：浮现“此页视点卑近，似非成稿”“宜配边压覆”等文字。

**核心谜题**：诗词比对。

**玩家操作**：

1. 左栏阅读画中题诗。

2. 右栏对照参考流传版本。

3. 逐字点击四处差异字。

4. 差异字组合为“画非一人”。

**题面答案**：

**获得物件**：残砚、旧批注残片。

**关键线索**：

- “画非一人”表层理解为“另一个人参与作画”。

- 深层理解为“这幅画里不只有一个人的观看”。

- 旧批注说明后人不是恶意毁坏，而是以统一体例、保留画心、压覆边旁杂线的方式遮蔽来源。

**章末结果**：返回现实后，周鹤年讲解版本比对与异文判断，提示“一个字是巧合，一句话是线索，一套证据链才是判断的基础”。

---

## 第三章 · 西园 · 卅六鸳鸯馆至留听阁

**主场景**：卅六鸳鸯馆、留听阁。

**章节目标**：完成误读反转，确认王蘅真正重要的不是画技，而是空间观看。

**关键场景**：

- 卅六鸳鸯馆北厅：散落园林草图，比例歪斜、线条笨拙，动摇“王蘅是天才女画家”的误读。

- 草图渗字：“看得到吗”。

- 留听阁：封住的墙面下有旧线凹凸。

**核心谜题**：草图显现。

**玩家操作**：

1. 打开物件匣，选择残砚。

2. 将残砚靠近墙面，用朱砂轻拂，灰泥表面浮现淡红色线条。

3. 选择断簪，沿红线轻划，灰泥沿旧刻痕剥落。

4. 墙面草图显现，点击右下角凹槽，打开暗格。

**获得物件**：王蘅的信、草图拓片。

**关键线索**：

- 墙面草图从芙蓉榭栏杆下的低处看去，将远香堂倒影、小飞虹弧线、梧竹幽居竹影纳入同一视野。

- 墙角题字：“吾笔拙，不能写形。然此处水桥竹影，三景同入一眼。知我者，当知我所见。”

- 王蘅信件核心：“将我放在园子里。不必有名，不必有形。只要有痕迹。”

**章末结果**：返回现实后，周鹤年承认自己年轻时也见过一个淡淡的“蘅”字，却没有写进报告。玩家意识到现代学术谨慎也可能再次遮蔽王蘅。

---

## 终章 · 第三十一景

**主场景**：画中世界 → 宣纸真相空间。

**章节目标**：复原第三十一景的观看来源，并让玩家承担最终解释选择。

**场景呈现**：

- 宣纸空白空间。

- 拙政园俯瞰线图浮现。

- 右侧悬着断簪、残砚、草图拓片。

- 左侧悬着“画非一人”“此页视点卑近，似非成稿”等线索。

**核心谜题**：四问复原。

**完成后真相**：

- 第三十一景是文徵明采用王蘅的草图、标记与观看位置，用自己的笔完成的画。

- 王蘅的贡献不是画法，而是发现了这个观看关系。

- 文徵明保存了她的眼睛，但没有把这双眼睛归还给她。

- 后人保留正式画心，却把来源痕迹排除在作品体系之外。

**最终回声**：

> 有人，终于看到了。
>
>

**章末结果**：出现三条路，进入结局选择。

# 结局系统（王子轩负责）

MVP 只在终章做结局选择，不做前置复杂分支树。前序线索影响提示完整度和结尾总结素材，但不阻断主线，也不改变结局入口。

## 结局一：存档

**立场**：历史正义。

**选项文案**：我会在修复报告里记下这个可能性。

**选择前犹豫提示**：她说过“不必有名，不必有形”。你确定要替她做一个她没有要求的决定吗？

**报告核心**：

> 第三十一景表层图像整体稳定，然页边装裱层下可见旧题签残痕与低位辅助线。结合画中视点、题诗异文及相关残迹，不排除该景保存了一位未被记录的观看提供者之视角。此判断仍需进一步考证。
>
>

**屏幕收束**：

> 有人看到了。
>
>

## 结局二：守密

**立场**：私人守护。

**选项文案**：有些痕迹，被看见就够了。

**选择前犹豫提示**：如果你不说，也许永远不会有人知道她存在过。历史会继续沉默。你能接受吗？

**报告核心**：

> 残页已修复，表层图像稳定。页边残痕暂不具备独立判断条件，留待后续观察。
>
>

**屏幕收束**：

> 有人来过。够了。
>
>

## 结局三：续笔

**立场**：当代续写。

**选项文案**：我想画一幅新的画。

**选择前犹豫提示**：你可以不替历史下结论，也不替她守住沉默。你只回应她的观看。但这幅新画，终究会带上你的解释。

**行动核心**：沈念画下芙蓉榭栏杆下的低位视角，并在右下角写：

> 续文徵明三十一景图。此为第三十二景。记王蘅所见。
>
>

**屏幕收束**：

> 三十一景之后，还有第三十二景。
>
>

# 谜题设计总表（刘子宣负责）

## 序章：扫描比对谜

**目标**：发现第三十一景画心稳定，但来源痕迹被装裱遮蔽。

**答案**：装裱接缝异常、背纸年代偏差、侧光下有“所见”残字与低位辅助线。

**玩家操作**：放大镜 → 纸质分析 → 侧光照射 → 点击残字与辅助线交会处。

**解开后获得**：修复笔记本，触发跌入画中。

**AI/提示**：

1. 不要先判断画风，先看边、背纸、接缝、压痕。

2. 表层画面完整，真正不完整的是说明来源的部分。

3. 侧光模式下“所见”残字和低位辅助线轻微闪烁。

## 第一章：水面倒影谜

**目标**：从倒影中取得现实栏杆上不存在的断簪。

**答案**：断簪只存在于倒影，需要镜像翻转后拾取。

**玩家操作**：点击倒影断簪 → 点击“镜像视角 / 翻转”按钮 → 拾取断簪。

**解开后获得**：断簪，记录“蘅”字。

**AI/提示**：

1. 倒影里有，实物却没有。

2. “倒过来的方式”不是谜语，是操作方式。

3. 屏幕边缘“翻转”图标闪烁。

## 第二章：诗词比对谜

**目标**：找出画中题诗与参考版本的四处差异。

**答案**：画、非、一、人。

**玩家操作**：左右栏逐字比对，点击差异字。

**解开后获得**：残砚，触发旧批注残片，确认“画非一人”。

**AI/提示**：

1. 逐字对照，不要凭记忆。

2. 四处差异都在实词位置，且可直接取字。

3. 差异字高亮，但组合含义仍由玩家理解。

## 第三章：草图显现谜

**目标**：显现留听阁封墙下的低位视角草图。

**答案**：先用残砚朱砂显出旧线，再用断簪沿线揭开灰泥。

**玩家操作**：打开物件匣 → 使用残砚 → 使用断簪 → 点击暗格。

**解开后获得**：王蘅的信、草图拓片。

**AI/提示**：

1. 墙面下不是正式画作，更像草图。想想谁留下了标线工具。

2. 残砚里的朱砂和墙面旧线同源。先让线显出来。

3. 线已经出现了。沿着线剥离灰泥，用她曾经划线的东西。

## 终章：第三十一景复原谜

**目标**：确认第三十一景的观看来源。

**答案**：

- 谁在看：王蘅，作为被遮蔽的观看者。

- 从哪里看：芙蓉榭栏杆下的低处。

- 看到了什么：远香堂倒影、小飞虹弧线、梧竹幽居竹影。

- 为什么来源被遮蔽：画面还在，但说明来源的边注、题签和草图痕迹被装裱遮住。

**玩家操作**：四问选择/地图点击/多选景物，答对后线稿逐步浮现。

**解开后获得**：三结局选择入口。

## 关键界面

左栏画中题诗，右栏参考版本。玩家点击差异字，差异字高亮，但不直接解释组合含义。

4. **草图显现界面**
墙面 \+ 物件匣。玩家先选残砚显红线，再选断簪揭灰，最后点击暗格取得信件。

# AI功能与状态设计（张欣悦负责）

## AI / 提示系统定位

当前前端已接入 DeepSeek V4 Pro，API Key 通过 `Web/.env` 中的 `VITE_DEEPSEEK_API_KEY` 配置，不得提交到仓库。固定规则提示仍作为基础兜底，确保关闭 AI 或未配置 API Key 时，玩家仍能完整通关。

已落地模块：

- `Web/src/core/ai-service.js`：DeepSeek API 统一封装，提供周鹤年对话、笔记本查阅、沈念批注、修复报告生成接口。

- `Web/src/core/ai-prompts.js`：AI System Prompt 模板，动态注入章节、世界、物件、进度、批注上下文。

- `Web/src/components/chat-panel.js`：周鹤年 AI 对话面板，仅现实世界显示。

- `Web/src/components/notebook-panel.js`：AI 笔记本面板，支持批注时间线与画中世界查阅。

AI / 提示系统只负责：

- 根据当前章节与进度变量提供渐进提示。

- 根据已获得线索回答有限范围问题。

- 在结局阶段生成或辅助呈现修复报告式总结。

AI / 提示系统不负责：

- 生成主线剧情。

- 决定谜题答案。

- 控制结局。

- 新增历史事实、人物关系或文物考证结论。

- 在玩家未获得线索前泄露真相。

关闭 AI 后，玩家仍必须能完整通关。

## 提示层级

每个谜题设 3 级提示：

1. 方向提示。

2. 关键线索。

3. 接近直给答案。

提示触发方式：

- 玩家主动点击修复笔记本提示。

- 玩家多次错误操作后自动升级。

- 玩家停留过久后可弹出轻量入口，但不强制。

## 存档数据说明

# MVP已定决策（重要，所有人经常核查）

1. 王蘅为虚构人物，不做历史考证式写法。

2. 核心真相是“文徵明的笔，王蘅的眼睛”，不是“找回失踪画作”。

3. 第三十一景画面一直存在，后人遮蔽的是来源痕迹。

4. 王蘅不是被埋没的天才女画家，而是被遮蔽的空间观看者。

5. 第二章诗词比对隐藏短句使用“画非一人”。

6. 终章结局三个都实现，MVP 用文本结局。

7. 固定规则提示作为基础兜底；DeepSeek LLM 已接入，用于周鹤年对话、笔记本查阅、沈念批注和修复报告生成。

8. 页面交互采用章节节点 \+ 点击调查 \+ 按钮选择。

9. 地图形式不做自由大地图，只做章节式节点跳转。

10. 第三十一景最终画面使用预制线稿/背景图 \+ 文本呈现，不依赖实时生成图。
