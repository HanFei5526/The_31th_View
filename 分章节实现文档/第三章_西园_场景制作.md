# 第三章 · 西园 — 制作文档

> **台词标记说明**
>
> - `【旁白】` = 叙事旁白，无说话人
> - `【周鹤年】` = 导师对话，显示说话人名
> - `【沈念·内心】` = 主角内心独白，用斜体样式区分
> - `【系统提示】` = 游戏机制提示
> - `【浮条】` = 底部浮现旁白条，4s 后自动消失，不阻塞操作
> - `【渗字】` = 场景内原位渲染，逐字从纸面/墙面渗出（300ms/字）
> - `【题字·强调】` = 场景内高亮渲染，字体加大
> - `【引用·强调】` = narrationBar 引用态，居中加引号样式
> - 每条台词为一次点击/自动推进的最小单位（浮条/渗字除外）

**3-8b**　【系统提示】也许物件匣中有能帮上忙的东西。

**3-9**　【旁白】你打开残砚，将砚中残余的朱砂靠近墙面。

**3-10**　【旁白】朱砂还没碰到灰泥，墙面就有了反应——淡红色的线条从灰泥下浮现出来，一条，两条，越来越多。同源的朱砂在画中世界里产生了呼应。

**错误操作：先选断簪**

**3-11**　【浮条】簪尖划过灰泥，没有反应。也许需要先让旧线显现出来。

**红线显现后再次点击墙面 → 选择断簪（正确第二步）**

**3-12**　【旁白】你取出断簪，沿着一条红线轻轻划过。

**3-13**　【旁白】灰泥沿旧刻痕簌簌剥落。簪尖的磨痕与墙上的旧线严丝合缝——她曾用这根簪，在未干的灰泥上划出这些定位线。

> 灰泥剥落动画（1.2s），背景图切换为 `chapter3-sketch-revealed.png`

**草图显现后叙事（自动播放）**

**3-14**　【旁白】灰泥一片片落下。墙面露出一幅画——不，与其说是画，不如说是草图。

**3-15**　【旁白】它很拙。远香堂画得太低，小飞虹弯得太急，竹影几乎压到画面边缘。线条有几处犹豫的地方，像画的人反复擦掉，又重新落笔。

**3-16**　【沈念·内心】这笔触……和北厅地上那些散落的草图太像了。同样的犹豫，同样的比例失调。但这一张没有被撕碎，没有被放弃——它被留在了墙上。

**3-17**　【系统提示】试试后退一步，蹲下来看。

**3-19**　【旁白】忽然，所有"不对"都对了。

**3-20**　【旁白】你想起刚才捡起的那张画——水面太重，桥线太弯，亭阁压得太低。那张画和眼前这幅草图的视角几乎一样。从这个高度看过去，远香堂的倒影、小飞虹的弧线、梧竹幽居的竹影，真的会同时出现在一个画面里。

**3-21**　【沈念·内心】她画得不好。但她看得很准。
