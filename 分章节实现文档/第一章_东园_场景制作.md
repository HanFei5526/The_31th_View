# 第一章 · 东园 — 制作文档

> **台词标记说明**
>
> - `【旁白】` = 叙事旁白，无说话人
> - `【周鹤年】` = 导师对话，显示说话人名
> - `【沈念·内心】` = 主角内心独白，用斜体样式区分
> - `【系统提示】` = 游戏机制提示
> - `【浮条】` = 底部浮现旁白条，4s 后自动消失，不阻塞操作
> - 每条台词为一次点击/自动推进的最小单位（浮条除外）

## 场景1 · 兰雪堂（阶段②）

### 背景图

| 必要元素 | ① 青石板路 ② 两侧翠竹 ③ 敞厅建筑 ④ 门楣匾额（"兰雪堂"三字清晰可辨） ⑤ 廊柱 |

### 台词

**入场旁白**

**1-1**　【旁白】你睁开眼。四周的一切都不对——不是工作室，不是扫描仪的冷光。自己站在一个陌生的地方。

**1-2**　【沈念·内心】这是哪里？

**1-3**　【旁白】低下头，脚下是青石板路，微微发暖。空气里有一种说不出的气味——不是花香，更像旧纸和松烟墨混在一起的味道。抬起头，两侧翠竹如墙，在没有风的空气中轻轻摇动。

**1-4**　【旁白】顺着石径看过去。前方有一座敞厅，像是某种入口建筑。

**1-5**　【沈念·内心】等等……这个地方我见过。是拙政园东部的兰雪堂。但不对，它不是我在照片里看到的样子。它更旧，更轻——像是一笔还没有干透的线。

**1-6**　【沈念·内心】不会吧。我在画里？

> 旁白结束后，场景进入自由探索状态。可点击元素出现微弱光点。

**环境元素点击反馈**

**1-7a**　【浮条】你踩了踩脚下的石板。有些温热，纹理清晰得像刚刻上去的。

**1-7b**　【浮条】你拨开竹叶。沙沙作响，但手上没有感到风。

**1-7c**　【浮条】你走到廊柱边，伸手碰了碰。木纹里藏着细密的墨线——这不是一根真正的柱子，它是一笔画出来的。

**匾额叙事（点击匾额触发）**

**1-8**　【旁白】你走近敞厅，抬头看向门楣上的匾额。

**1-9**　【旁白】三个字——「兰雪堂」，落款是文徵明。字迹端正，墨色沉稳。

**1-10**　【旁白】你本想移开视线，但不知为什么，目光停住了。你又看了一眼。

**1-11**　【沈念·内心】「兰」字的草字头……下面好像多了一道横笔。很细，几乎要看不见。

**1-12**　【旁白】你凑近了一些。那一笔确实在那里——笔力很稳，墨色和主体一致，不像是无意的败笔。但它又不能单独解释成一个字。

**1-13**　【沈念·内心】奇怪。先记下来吧。

**1-14**　【系统提示】已记录线索：匾额多余笔画

> 匾额叙事结束后，前方出现方向指示。

**1-15**　【沈念·内心】这座厅后面还有路。石径延伸过去，像是有什么在更深处等着。

**2-5**　【旁白】缝隙里忽然露出远处水面的一线光——站着的时候完全看不见的光。

## 场景3 · 芙蓉榭 · 水面倒影谜题（阶段④⑤）

### 背景图

| 必要元素 | ① 芙蓉榭主体建筑 ② 栏杆（清晰可辨） ③ 水面（占画面下半，倒影清晰） ④ 倒影中栏杆上有一支模糊的断簪        |

| 尺寸     | 1920×1080，16:9                                                                                             |

### 台词

**入场旁白**

**3-1**　【旁白】你循着水声走出石径。眼前豁然开朗——一座临水小榭安静地立在岸边，屋檐低垂，栏杆探入水面。

**3-2**　【沈念·内心】芙蓉榭。拙政园东部最靠近水面的地方。

**3-3**　【旁白】你走到栏杆前，往下看。水面很静，倒影清晰得像另一个世界——每一根栏杆、每一片瓦都被原样映在水里。

**3-4**　【旁白】但看了一会儿，你发现不对。倒影里的栏杆上，好像多了什么东西。

**3-5**　【沈念·内心】上面什么都没有。可水里……水里的栏杆上挂着一件东西。

> 入场旁白结束后，进入倒影谜题交互。

**点击真实栏杆（未命中）**

**3-6**　【浮条】你摸了摸栏杆。什么都没有。但水面的倒影里，那件东西还在。

**点击水面线（断簪识别前）**

**3-6b**　【浮条】水面微微晃动，什么也没发生。

> 水面分界线从场景加载起即有微弱光泽波动（场景元素，非UI按钮），玩家随时可点击。断簪识别前点击只给氛围反馈。

**点击倒影中的断簪**

**3-7**　【旁白】你把目光压低，盯着水面里那件东西。涟漪轻轻散开，它的轮廓逐渐清晰。

> 水面涟漪动画，物件变清晰。此时水面线交互解锁（点击将触发翻转）。

**3-8**　【旁白】是一支簪子。簪头是半朵芙蓉的形状，断了，断口处已经发黑。它静静地挂在倒影的栏杆上。

**3-9**　【沈念·内心】只在倒影里存在的东西……我怎么把它拿出来？

**再次点击倒影中的断簪（可选，额外提示）**

**3-10**　【浮条】你试着伸手去够水面，但指尖一碰，倒影就散了。

**点击水面线（断簪识别后）**

**3-11**　【沈念·内心】如果倒影才是真的那一面呢——

> 画面翻转动画（0.8s）。

**3-12**　【旁白】世界颠倒了一瞬，又重新安定。倒影变成了正像，断簪就在你面前，触手可及。

**拾取断簪**

> 点击翻转后的断簪，播放拾取动画。

**3-13**　【旁白】你伸手，取下了那支断簪。银质已经发暗，簪头是半朵未开的芙蓉，做工细致。

**3-14**　【旁白】你下意识翻过来看簪身背面——一个极小的字，刻在不显眼的位置。

**3-15**　【沈念·内心】蘅。

**3-16**　【沈念·内心】刻在背面，这么小……不像题名，也不像工匠的标记。那是什么？

**3-17**　【系统提示】获得物件：断簪

> 物件描述：银质断簪，簪头为半朵芙蓉。簪身背面刻有一个极小的"蘅"字。不像正式题名，也不像工匠标记，用途不明。它无法带出现实，却能在画中划开同样属于王蘅的痕迹。

**3-18**　【系统提示】已记录线索：断簪"蘅"字刻痕

## 场景3b · 轻量讨论（阶段⑤）

### 视觉

不需要额外背景图。仍使用芙蓉榭场景（翻转已复原为正常视角），笔记本面板展开为焦点。

### 笔记本预置批注

> *（周老师的批注）"蘅"，杜衡。是一种香草，古人也用来比喻品性高洁的女子。刻在簪身背面，不是正面——如果是工匠标记，通常会在簪头或底座。这更像是物主自己留给自己的。那么，是谁把自己的名字藏在了一支断簪上？*

### 快捷按钮

> 对话框上方单独设置「跳过讨论」按钮。

- "也许是这支簪子主人的名字"
- "为什么要藏在背面？"

### 讨论结束后

**3-19**　【旁白】水面重新归于平静。你把断簪小心地收起来。

**3-20**　【沈念·内心】蘅。按周老师的批注，这个字有女子名的意味。

**3-21**　【沈念·内心】它偏偏刻在簪身背面，又小得几乎要错过。像是刻字的人想留下一点痕迹，却仍把它收在不显眼处。

**3-22**　【旁白】你站在榭前，忽然觉得眼前的光线在变。周围的墨色好像在一点点褪去，像清水洗过宣纸。

### 图片风格统一要求

第一章所有画中世界背景图应遵循以下风格规范：

| 要素     | 规范                                         |
| -------- | -------------------------------------------- |
| 色调     | 暖旧色调，偏米黄/浅赭，模拟宣纸底色          |
| 笔触     | 水墨淡彩风格，非写实，线条可见但不生硬       |
| 纹理     | 底层叠加淡纸纹肌理，模拟"身在画中"的观感     |
| 光线     | 永恒午后光感，不做强烈明暗对比               |
| 整体感受 | 这不是一张园林照片，而是"一笔还没有干透的线" |
