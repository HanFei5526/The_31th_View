# 人物立绘设计

## 美术风格定位

**不是**动漫立绘，**不是**写实照片，而是介于两者之间的——

> **新工笔人物画风格**：线条清晰可见但不僵硬，设色淡雅（淡彩/浅绛），面部五官细腻但留有笔触感，背景留白或极简水墨晕染。参考艺术家气质：何家英工笔人物 × 现代概念插画的克制感。

---

## 统一规范

| 要素 | 规范 |
|------|------|
| 背景 | 纯白/米白/极淡水墨晕染，不画具体场景（立绘要叠加在游戏 UI 上） |
| 色调 | 沈念偏冷白灰（现实世界），周鹤年偏暖灰（学者温度），王蘅偏赭黄（画中世界底色） |
| 笔触感 | 关键——要能看到"画出来的"质感，区别于照片和赛璐珞动画 |
| 反面参考 | 禁止：大眼萌系、浓艳古风、飘带仙气、欧美奇幻、像素风 |
| 输出格式 | PNG，透明背景，分辨率不低于 1024×1536（竖版）或 1536×1024（横版） |

### 构图规格

| 类型 | 比例 | 裁切范围 | 用途 |
|------|------|----------|------|
| 全身 | 2:3 竖版 | 头顶至脚尖，留天地空间 | 角色介绍、立绘展示 |
| 半身 | 2:3 竖版 | 头顶至腰胯，人物居中偏左 | 对话框立绘（主用） |
| 侧身 | 2:3 竖版 | 全身或半身的侧面角度 | 回忆场景、过场 |

### 通用风格后缀（所有提示词末尾统一附加）

```
Style: modern gongbi painting meets editorial illustration, visible fine brushstrokes, light ink-wash coloring (淡彩), subtle rice-paper texture, elegant negative space. NOT anime, NOT photorealistic, NOT cel-shaded. --s 400
```

## 1. 沈念（玩家角色）

### 形象设定

- 25岁女性，瘦削文静，气质偏学院派内敛
- 短发或低马尾（方便工作），碎发微挡眼角
- 穿白大褂（修复室工装）或素色棉麻衬衫，袖口卷起
- 手指修长偏瘦，指甲短而干净（修复师特征）
- 颈间无饰品，耳后可能夹一支铅笔
- 身材偏瘦，中等身高，站姿微微前倾（习惯性观察姿态）

## 2. 周鹤年（导师）

### 形象设定

- 约60岁男性，清瘦儒雅，银白短发梳理整齐
- 面容有岁月纹路但精神矍铄，目光锐利而温和
- 常着深灰/藏青色中式立领衫或旧款西装外套
- 手持放大镜或轻拈毛笔的姿态
- 气质：老派学者的严谨与淡然，"不轻易下结论"的克制感
- 薄框眼镜（阅读用，可能推到额头上）
- 身材清瘦，偏高，站姿端正有学者风度

## 3. 王蘅（回声/痕迹——仅供内部参考）

> **重要：游戏内不会出现王蘅人形立绘**，她只以痕迹存在（断簪、小砚台、墙壁草图、渗出残字）。以下仅作为内部美术参考，用于确定与她关联物件的气质方向。

### 形象设定

- 明代中期年轻女性，约18-20岁
- 穿着简素的交领衫裙（非华贵），布料质朴，袖窄便于书写作画
- 发髻松散（非正式盘发），以簪固定——那支断簪刻有"蘅"字
- 手指有墨渍和朱砂残留
- 表情不悲不喜，是一种长期观察者的安静专注
- 没有"仙女飘逸感"，是扎实站在地面上看园子的人
- 身材娇小偏瘦，赤足或穿布履

## 生成注意事项

1. **一致性控制**：同一角色不同表情/角度的生成建议使用相同 seed 基础或 character reference，确保五官、发型、体型在各张之间保持一致。
2. **背景统一去除**：所有立绘最终需要透明背景，生成时用纯白/米白背景，后期抠图。
3. **批量命名规则**：`角色名_构图_编号.png`，如 `shennian_halfbody_B2.png`、`zhouhenian_fullbody_A1.png`。
4. **优先级**：半身立绘（B系列）为对话框主用，优先生成；全身（A系列）用于介绍和过场，可后续补充。
5. **表情微调**：如需同一构图的更多表情变体，在对应提示词基础上只修改表情描述部分，保持服装和构图不变。
