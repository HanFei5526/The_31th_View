# 综合研讨提示与浅纸色转场统一

## 修改目标

- 补充综合研讨阶段的系统提示，让玩家知道得出结论后需要点击“确认这个推断”按钮继续。
- 将现有偏黑、偏浓黄的转场遮罩调整为更接近参考图的浅白偏纸黄色。
- 保留原有叙事节奏和转场功能，不改研讨判定逻辑。

## 涉及文件

- `Web/src/pages/prologue.js`
- `Web/src/styles/transitions.css`
- `Web/src/core/game-engine.js`
- `Web/src/styles/index.css`

## 主要改法

- 修改综合研讨开始时的系统提示文案，明确提示“得出结论后点击确认这个推断即可继续”。
- 将通用淡入淡出、宣纸晕染、跌入画中终段、世界切换遮罩统一改成浅纸色半透明质感。
- 将褪色转场的实色背景从偏黄宣纸色调浅，避免章节过渡时出现黄底闪屏。

## 完成标准

- 玩家进入综合研讨时能明确知道后续需要点击确认按钮。
- 跌入画中和章节/世界切换时不再出现明显黑色遮罩。
- 返回或褪色类转场不再出现浓黄底色，整体接近“发白但带一点纸黄”的参考色。
- `npm run build` 通过。
