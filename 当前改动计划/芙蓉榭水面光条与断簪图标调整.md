# 芙蓉榭水面光条与断簪图标调整

## 修改目标

- 将芙蓉榭水面的可点击光条下移，使它更接近画面中的水面位置。
- 将光条颜色从白色改为淡黄色，与当前光点的金黄色调保持一致。
- 将断簪图标从具象 CSS 小图形改为更抽象、克制的线条 SVG。

## 涉及文件

- `Web/src/pages/chapter1-paint.js`
- `Web/src/styles/index.css`

## 主要改法

- 调整 `.furong-waterline` 对应元素的 `top`，让点击光条更贴近水面。
- 改写 `.furong-waterline` 的渐变色与发光效果，使用淡黄/金色透明层。
- 在断簪 DOM 中加入抽象线条 SVG，并移除原先的具象 CSS 伪元素绘制。

## 完成标准

- 光条位于水面附近，视觉上不再悬在建筑上。
- 光条呈淡黄色，不再是白色。
- 断簪显示为抽象线条图标，整体更雅致，不像具象 emoji 或粗糙小图。
- 前端构建通过。
