# 第三章匾额回看素材显示调整方案

## 问题现象

第三章回看匾额时，当前直接显示完整的 `chapter1-lanxuetang-plaque.png`，画面范围过大，屋顶、门扇等信息占据太多注意力。玩家此处真正需要看的，是匾额和周围少量结构。

## 调整目标

1. 将匾额回看画面缩小到更接近截图红框标注的范围。
2. 画面只保留匾额附近的局部信息，弱化屋顶和下方门扇。
3. 整体提亮，让它更像回忆中的视觉片段。
4. 显示时加入透明度，避免压住留听阁背景和沈念立绘。

## 实现方式

1. 匾额回看不再直接复用草图近景的完整图片显示方式。
2. 在 `chapter3-paint.js` 中给匾额图片使用独立 class。
3. 在 `chapter3.css` 中为 `.ch3-plaque-panel` 设置更小的宽高、裁切范围、透明度和亮度滤镜。
4. 不改动原始图片素材文件，方便后续继续复用。

## 涉及文件

- `Web/src/pages/chapter3-paint.js`
- `Web/src/styles/chapter3.css`

## 验收方式

1. 回看匾额时，画面尺寸明显小于当前完整大图。
2. 可见内容集中在匾额及其附近区域。
3. 匾额图比原图更亮，并带有轻微透明感。
4. 留听阁背景仍能透出，不遮挡底部叙事框。
