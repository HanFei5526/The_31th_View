# 返回主页黄底闪屏修复

## 修改目标

- 修复从任意章节返回主页时短暂露出的偏黄色空白背景。
- 保留必要的场景过渡感，不让返回动作显得突兀。
- 将过渡底色调整为更白、更轻的宣纸感，并淡化纹理观感。

## 涉及文件

- `Web/src/core/scene-manager.js`
- `Web/src/core/game-engine.js`
- `Web/src/pages/menu.js`

## 主要改法

- 在场景管理器里识别返回 `menu` 的场景切换，改走返回主页专用转场。
- 在引擎里新增轻白色的返回主页遮罩，避免使用当前偏黄的通用遮罩。
- 调整主页场景首帧背景色，让菜单背景图加载或入场动画期间不再露出明显黄底。

## 完成标准

- 从章节点击返回主页时不再出现明显黄色空白页。
- 过渡颜色更接近发白的淡宣纸色，纹理感更弱。
- 普通章节内切换和已有章节过场不被破坏。
- `npm run build` 通过。
