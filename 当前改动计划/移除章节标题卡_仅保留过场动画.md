# 移除章节标题卡：仅保留过场动画

## 问题

当前序章和第一章开头存在独立标题卡，卷轴转场组件中也会创建章节标题卡。后续章节不需要标题卡，只需要相应的过场动画。

## 修改范围

- `Web/src/pages/prologue.js`
- `Web/src/pages/chapter1-paint.js`
- `Web/src/components/transition.js`
- `Web/src/styles/transitions.css`
- `Web/src/pages/game-scene.js`

## 修改方式

- 删除序章开头标题卡调用和实现。
- 删除第一章画中世界开头标题卡调用和实现。
- 删除卷轴转场中的章节标题卡 DOM，只保留卷轴合拢/打开动画。
- 删除通用场景基类中的章节标题 DOM、方法和样式。
- 保留序章跌入画中的 `FallTransition` 动画和其他真实转场动画。

## 验收标准

- 源码中不再存在 `_playTitleCard`、`transition-chapter-card`、`_showChapterTitle` 等标题卡入口。
- `FallTransition` 相关序章过场动画仍存在。
- `npm run build` 通过。
