# 第三章草图显现节奏调整方案

## 问题现象

第三章留听阁中，玩家使用断簪剥落墙面后，场景背景会切换到已揭露墙面状态，同时又立即叠加一张草图近景图。这样会让“墙面草图已经露出”和“草图近景显现”同时发生，节奏过快，信息重复。

## 调整目标

1. 让玩家先在房间整体语境里看到墙面被揭开。
2. 草图近景后移到“蹲下观察”阶段，作为低位视角验证的视觉焦点。
3. 保持原有证据链不变：残砚显朱砂线，断簪剥灰泥，蹲下验证低位视角，随后题字、拓片、暗格继续推进。
4. 不改动核心剧情基准。

## 新流程

1. 初始留听阁使用 `chapter3-liutingge.png`。
2. 残砚使用成功后，只叠加红线层，不切换背景。
3. 断簪使用成功后，播放灰泥剥落动画，切换到 `chapter3-liutingge-revealed.png`，此时只显示整面墙上的草图。
4. 自动播放草图显现叙事，玩家理解这是一张拙朴草图。
5. 出现“蹲下观察”按钮。
6. 玩家点击“蹲下观察”后，再淡入 `chapter3-sketch-revealed.png` 作为草图近景，并配合低位观察叙事。
7. 低位验证完成后，草图近景保留，继续承接题字、拓片与凹槽流程。

## 涉及文件

- `Web/src/pages/chapter3-paint.js`
- `Web/src/styles/chapter3.css`

## 验收方式

1. 使用残砚后只看到红线，不出现草图近景。
2. 使用断簪后切换到已揭露墙面背景，但不出现居中的草图近景。
3. 草图显现叙事播放完后，出现“蹲下观察”按钮。
4. 点击“蹲下观察”后，草图近景再淡入并进入低位验证叙事。
5. 后续题字、草图拓片、凹槽、信件流程不受影响。
