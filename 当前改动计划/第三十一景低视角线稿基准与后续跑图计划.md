## 当前线稿基准

- 基准图：`TestPic/finale-view31-line-low-angle-draft-4.png`
- 生成时间：2026-06-21
- 当前用途：作为第三十一景低视角画心线稿基准，用于后续同构设色层、完成态画心和遮蔽扫描件的视觉来源。

## 已确认视觉方向

1. **视角**
   - 低视角，从亭/廊内靠近栏杆的位置看向园林水面。
   - 有轻微仰视感，屋顶和梁架需要露出较多。
   - 前景栏杆、地板、屋顶梁架的透视趋势应统一，不能局部突然反向。

2. **构图**
   - 左侧为亭内柱、廊檐、木地板与栏杆。
   - 中左为远香堂式建筑与树丛。
   - 中部为水面、倒影、小桥、远山。
   - 右侧为竹丛与水岸。
   - 荷叶位置可接受在前景右下或水面近处，不再作为必须回到左下的硬约束。

3. **风格**
   - 黑白/灰阶线稿，宣纸旧色底。
   - 线条细密度以当前基准为上限，后续资产不得比当前基准更细、更密或更碎。
   - 保持古画修复、建筑测绘与文人画线描之间的气质。
   - 不要现代 UI、题字、人物、锦边、装裱边框。

4. **硬性保留**
   - `finale-view31-line-low-angle-draft-4.png` 的低视点、屋顶露出和前景栏杆透视作为后续主基准。
   - 后续任何彩色层或完成态的画心都必须与该图同构图、同裁切、同视角，避免终章填色阶段错位。

### 第二步：基于线稿生成设色层

目标：
- 从当前线稿生成同构彩色画心。
- 保留所有建筑、栏杆、竹丛、桥、水面、远山位置。

约束：
- 不加锦边。
- 不加题字。
- 不加扫描污渍。
- 不改变视角。
- 不移动前景栏杆和屋顶。
- 颜色应偏淡雅古画设色，不要鲜艳插画风。

验收：
- 与线稿叠加时主体结构基本对齐。
- 局部填色时不会出现明显错位。
- 色彩服务“修复完成”，不是独立概念图。

### 第四步：制作遮蔽扫描件

目标：
- 制作一张“带装裱层的旧扫描件”，用于序章/前期被遮蔽状态。

约束：
- 必须带锦边、边框、题字区或装裱层。
- 遮蔽信息应体现在边注、题签、残字、辅助线、扫描痕迹、旧污损上。
- 不能做成单独画心上随意叠污渍。
- 不得出现清楚可读且误导剧情的大字题名。

验收：
- 玩家能看出这是被扫描/归档过的装裱件。
- 玩家能看出关键来源信息被遮蔽，而不是画面本身缺失。
- 与终章完成态存在同源关系。

```text
Use the provided monochrome line drawing as the exact composition and perspective reference.

Preserve every architectural line, railing, roof beam, bridge, bamboo grove, hall, water reflection, lotus, and mountain position.
No mounting border, no brocade border, no title inscription, no scan damage.
Keep pale mineral colors, aged paper, restrained literati painting mood.

```

### 完成态提示词重点

```text
Use the colored painting-heart as the unchanged center image.
Add elegant album-leaf mounting, brocade or silk border, aged paper backing, subtle conservation-complete presentation.

```

### 遮蔽扫描件提示词重点

```text
Create an archival scan of the mounted album leaf using the same painting-heart.
Include brocade border, mounting paper, edge frame, label/title area, old stains, scanning shadows, blocked marginal notes, obscured auxiliary lines, covered remnants, and partial residue.
The obscuration should affect the marginal/source traces, not randomly cover the painting-heart.
No clear false title, no modern UI, no people.
```

## 风险点

1. 图像生成容易重构外景，后续必须反复强调“同构图、同裁切、同视角”。
2. 设色层如果与线稿错位，会影响终章局部填色效果。
3. 完成态如果改变画心位置，会造成终章转场跳变。
4. 遮蔽扫描件容易被生成成裸画心污损图，必须明确“带装裱扫描件”。
5. 局部建筑透视容易出现小段反向，尤其是屋顶椽头、栏杆和柱前结构，后续需要放大检查。
