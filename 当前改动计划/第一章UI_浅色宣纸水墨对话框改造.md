# 第一章UI：浅色宣纸水墨对话框改造

## 1. 目标
按照用户需求，针对画中世界（第一章）的 UI 组件（底部叙事对话框），将其原本的暗木色风格改造为贴合“水墨淡彩、暖旧色调”的浅色水墨宣纸主题。

## 2. 依据约束与注意事项
- 遵循 `总分工文档/前端_交互_背包设计.md` 中的通用布局约束：叙事对话框（底部）宽度必须维持 **55%** 且居中，不可与右侧 360px 笔记本面板发生遮挡或位移。
- 遵循 `AGENTS.md`：正式图片必须存放在 `Web/public/images/` 目录下。
- 遵循 `AGENTS.md`：改动实施前必须创建计划文档，经过审批。（注：本次为补齐审批流程）。

## 3. 具体修改方案

### 3.1 叙事对话框样式（narration-bar.css）
- 新增 `.paint-world` 命名空间下的样式覆盖，确保浅色水墨主题仅在画中世界生效。
- **对话框底色与边框**：使用半透明的暖米色/深赭色搭配 `backdrop-filter: blur() sepia()` 形成类似古画宣纸的柔和半透明质感，边框改为通透的淡金棕色。
- **右下角纹饰**：在对话框右下角 `::before` 插入用户提供的芙蓉花纹样（去除背景后提取的纯红色线稿，存放于 `public/images/furong-icon.png`），并设置低透明度，形成宣纸上的红色印泥暗纹效果。
- **名牌图标**：在名牌文字 `narration-speaker` 旁添加同款芙蓉花纹样，透明度较高，保持清晰。
- **宽度修正**：严格保证 `.paint-world .narration-container` 的 `width: 55%`，修复此前由于 85% 宽度设置导致的潜在面板遮挡问题。

### 3.2 图像资源处理
- 编写后台脚本读取 `TestPic/` 或原上传中的芙蓉花线稿素材（`chapter1-furong.jpg`），去除白灰纸张背景，提取红色线条轮廓，输出为带有透明通道的 PNG。
- 将输出的透明线稿命名为 `furong-icon.png`，存放至 `Web/public/images/` 目录，供前端 CSS 引用。

## 4. 验证方式
- 浏览器中打开第一章，确认对话框呈现浅色宣纸水墨主题。
- 确认对话框宽度为 55%，打开右侧笔记本面板时两者不重叠。
- 确认对话框右下角和名牌旁的芙蓉花纹样为红线透明底，自然融入背景。
