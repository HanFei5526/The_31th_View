# 确认推断按钮移到画布与对话框之间

## 修改目标

- 将“确认这个推断”按钮从讨论/笔记本对话框内部移到画布与底部对话框之间的空隙。
- 按钮视觉改为和左上角“返回”按钮一致的木刻印刷风格。
- 按钮尺寸保持克制，不遮挡画布主体、底部叙事对话框和右侧笔记本。

## 涉及文件

- `Web/src/components/notebook-floating.js`
- `Web/src/styles/notebook-floating.css`
- `Web/src/styles/gate-panel.css`

## 主要改法

- 保留现有确认按钮逻辑，只调整确认区域的挂载位置和定位方式。
- 将 `.notebook-confirm-area` 做成固定浮动条，放在底部对话框上方的中间偏下区域。
- 将 `.notebook-confirm-btn` 和旧 `.gate-confirm-insight-btn` 统一成返回按钮风格。
- 加入窄屏位置约束，避免和右侧笔记本、底部对话框互相压住。

## 完成标准

- “确认这个推断”出现时位于画布与对话框之间。
- 样式与返回按钮同源：木色底、厚阴影、金色文字、较小尺寸。
- 按钮确认后仍能正常触发原有回调。
- `npm run build` 通过。
