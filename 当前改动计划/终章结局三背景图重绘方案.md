## 一、图片定位

这张图只在终章选择“续笔”后作为结局三背景出现。它承担的是沈念新画的视觉落点，对应“三十一景之后，还有第三十二景”。

结局三的核心是用当代创作回应王蘅的观看，同时保持学术边界。第三十一景已经保存了她看见的园；结局三要呈现的是“看见园的她”。这幅图必须明确带有沈念的手感：她和王蘅一样并非成熟画家，画得不必精致，重要的是把人、位置和观看留下来。

因此重绘后画面应表达：

- 王蘅作为“被遮蔽的观看者”第一次获得一个轮廓，但这个轮廓来自沈念笨拙而郑重的续画。
- 她仍然很小、很克制，不成为英雄式主角。
- 她蹲在芙蓉榭栏杆下，从低处观看园林。
- 袖口有朱砂，身前有比例失准的草图，呼应第三章证据链；人物和草图都可以生涩、歪斜、线条不稳。
- 水面、远香堂轮廓、小飞虹弧线、竹影仍然可辨，承接第三十一景的低位视角。
- 画面情绪是安静、迟来的回应，避免胜利、昭雪或灵异显形的气质。

## 二、当前图片问题

当前图是一张明亮园景插画，优点是干净、可读、16:9 适配正常，但与结局三的叙事信息不够贴合：

- 只有园林，没有“蹲在栏杆下观看的人”，缺少“第三十二景画看见园的她”的核心。
- 视角更像常规平台远眺，低位观看感不强。
- 画面偏现代精致插画，也过于成熟，和“沈念画出第三十二景”的设定不够吻合。
- 没有朱砂袖口、拙劣草图、新纸、小字等续笔信息。
- 右下角疑似有生成痕迹或水印样高光，需要在新图中彻底避免。

## 三、重绘方向

### 3.1 画面内容

建议画成“沈念新画中的场景”，减少现实拙政园照片感。

主构图：

- 前景左下或中下：芙蓉榭木栏杆下方，一个年轻女子蹲坐或半跪，背对观者，侧着头看向水面与远处园景。
- 她的体量要小，约占画面高度的 10% 到 16%，不要正脸，不要强烈戏剧动作，人物线条可以略显笨拙。
- 她袖口有一点旧朱砂色，身前铺着一张草图，纸上有不稳的桥线、水面线和亭阁轮廓；草图应像不会画画的人努力记录位置，不能画得漂亮。
- 中景：栏杆、水面、荷叶或浅浅倒影，强调她的观看高度很低。
- 远景：远香堂轮廓、小飞虹弧线、右侧竹影，位置关系继承 `finale-painting-complete.png` 和 `finale-view31-low.png`。
- 右下角可有一行极淡小字“记王蘅观园”，但如果生成文字不稳定，优先不让模型直接生成清晰文字，后续可用前端或后处理加字。

### 3.2 视觉风格

- 低饱和纸本淡彩或淡墨线稿，线条允许歪斜、迟疑、重复试线，色彩点到为止。
- 继承终章完成图的宣纸质感、淡青绿水面、茶褐木构、灰绿竹影。
- 笔触要明显比正式第三十一景笨拙，因为这是沈念的“续笔”。它可以不漂亮，但需要真诚、可辨。
- 王蘅本人的轮廓要柔和、含蓄，像被小心画出来的人，避免幽灵感、NPC 立绘感和精致人物设定图感。
- 整体明度保持清透，避免黑雾、暗角、强烈逆光和商业插画式高饱和。

### 3.3 叙事边界

必须避免：

- 把王蘅画成文徵明并列作者、成熟女画家或公开受封的人物。
- 让画面暗示第三十一景由王蘅亲手绘制。
- 出现正脸肖像、神化光环、鬼魂透明影子、明显灵异特效。
- 把“续笔”处理成历史事实或正式档案图像。
- 让右下小字变成“正名式署名”。它只能是沈念新画的题记口吻。

## 四、资产策略

建议先不覆盖正式文件，按以下顺序推进：

1. 在 `TestPic/finale-ending3-redraw/` 生成 2 到 3 张候选图。
2. 选择最贴近叙事的一张，必要时做一次定向返工。
3. 确认后导出正式候选：`Web/public/images/finale/finale-ending3-garden-v2.png`。
4. 实机查看结局三播放和结局卡片遮挡效果。
5. 最终确认后再替换原 `finale-ending3-garden.png`，或改代码引用 v2 文件。

尺寸建议：

- 正式图优先使用 1672×941，和终章其他完成图保持一致。
- 若生成工具更适合 16:9，可先出 1536×864 或 1920×1080，再裁切/缩放到项目统一比例。
- 当前 CSS 使用 `background-size: cover; background-position: center;`，人物不能贴近四边，需保留足够安全边距。

## 五、首轮生成提示词草案

用途：历史叙事游戏终章结局背景图，16:9 横图。

画面：一张当代修复研究生续画出的笨拙纸本淡彩园林图。画者并不擅长绘画，只是郑重地记录一个人和一个观看位置。明代苏州园林芙蓉榭栏杆下，一个年轻女子背对观者蹲坐在低处，侧着头看向水面。她很小，园林很大。袖口沾有一点暗朱砂色，身前铺着一张比例失准的草图，纸上可见不稳的桥线、水面线和亭阁轮廓。水面映出远香堂的轮廓，中远景有小飞虹的弧线，右侧有梧竹幽居方向的竹影。整体从栏杆下的低位视角观看，强调“看见园的人”。

风格：低饱和纸本淡彩或淡墨线稿，宣纸纹理，少量淡青绿水面、茶褐木构、灰绿竹影。线条不必精致，可以有歪斜、重复、迟疑和擦改感；人物也不必好看，像沈念努力画出来的纪念画。画面清透明亮，避免照片感。

构图：宽幅 16:9，人物放在前景偏左下或中下，保留中央和右侧给水面、桥、竹影；不要让人物被结局卡片完全遮挡；画面边缘留安全区。

避免：正脸肖像、精致人物设定图、成熟古风插画、神化光环、鬼魂透明效果、现代服饰、现代建筑、商业插画高饱和、黑色粗描边、过度写实、文字乱码、水印、logo、印章、大段可读文字。

文字：不要生成清晰题字。右下题记“记王蘅观园”后续如有需要用后处理添加。

## 六、验收标准

- 缩略图状态下能一眼看出：有人在栏杆下低处观看园林。
- 画面中能辨认王蘅、草图、朱砂袖口、水面、远香堂倒影、小飞虹弧线、竹影这几类关键信息。
- 与 `finale-painting-complete.png` 有视觉关联，但完成度明显更低，能看出这是沈念续画的新纸。
- 与结局三文案互相补足，尤其是“第三十二景画了看见园的她”。
- 不破坏三条剧情基准：画心仍是文徵明完成的低视角成稿；王蘅仍是被见证而非被正式正名；她的出现属于沈念续笔的回应，不能成为历史定论。
- 结局卡片出现时，背景仍能提供情绪和信息，人物不会被完全遮住或与文字区域冲突。
