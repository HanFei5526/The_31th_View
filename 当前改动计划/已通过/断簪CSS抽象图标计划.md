# 断簪 CSS 抽象图标计划

状态：已通过（按用户当前指示直接实施）

## 目标

参考用户提供的簪子样式，将物件匣中“断簪”的图标和第一章场景中的断簪本体，从现有简单线段改为更像发簪的 CSS 抽象图标。后续按反馈收敛为极简版本：只保留一根完整斜向直线簪杆和顶部三个较小圆形，统一使用与笔记本图标一致的金色；簪杆略向左上移动，接近三个圆环左下交会处，并把线条调细到接近笔记本图标。最新反馈要求圆环只再缩小一点，簪杆终点贴住圆环边缘，不脱离也不穿出；若有视觉空隙，簪杆可轻微接入圆环边缘，并让圆环层级盖在上方。

## 改动范围

- `Web/src/components/inventory-popup.js`
  - 将 `hairpin` 物件图标从内嵌 SVG 改为 CSS 结构。
- `Web/src/styles/inventory-popup.css`
  - 新增断簪专用 CSS 样式，使用斜向簪杆和顶部三个小圆构成抽象图形。
- `Web/src/pages/chapter1-paint.js`
  - 将场景内倒影断簪与翻转后的断簪本体改为同一套 CSS 结构。
- `Web/src/styles/index.css`
  - 新增第一章场景内断簪 CSS 样式，并保留原发光提示、点击区域和翻转修正。

## 设计要求

- 只改断簪视觉，不改剧情、物件名称、物件描述、交互状态和背包逻辑。
- 图标参考图片中的长斜簪和头部三个圈，但保持简洁抽象；不保留额外珠点、连线、内圈或复杂装饰。
- 颜色沿用物件匣金色系，适配画中世界的暖纸背景。
- 不使用新图片资源，不引入额外依赖。

## 验收标准

- 断簪图标在 80px 物件格内清楚可辨。
- 第一章场景中的断簪仍可点击，倒影态和翻转后显示位置不变。
- 样式比原先几根直线更接近簪子轮廓，同时比例克制，顶部三个圈较小，整枚断簪只使用一种金色。
- `npm run build` 通过。
