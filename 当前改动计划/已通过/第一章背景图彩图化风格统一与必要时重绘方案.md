# 第一章背景图彩图化风格统一与必要时重绘方案

## 背景

第一章当前三张画中背景整体仍偏“淡墨旧稿”：纸底偏灰黄，植物、建筑、水面、石头的颜色层不够明确，画面完成度与第二章存在明显差异。

第二章当前背景更接近“手绘彩图”：纸色更亮更暖，木构、植被、水面、远景都有明确但不刺眼的颜色分层，整体更像已经完成设色的园林画面。

因此本次目标不是轻微调色，而是把第一章三张画中背景统一到第二章的手绘彩图风格。优先尝试在原图基础上做强风格编辑；若效果仍像灰稿或局部补色痕迹明显，则允许按原构图整张重绘。

## 修改范围

- `chapter1-lanxuetang-plaque.png` 为牌匾细节图，并在第三章中作为独立资源引用，本轮不纳入背景统一处理。

## 目标风格

以当前第二章背景为主要风格基准：

- `Web/public/images/chapter2/transition-chapter2.png`
- `Web/public/images/chapter2/chapter2-yuanxiangtang.png`
- `Web/public/images/chapter2/chapter2-xiaofeihong.png`

视觉目标：

- 第一章三张图要从“黑白灰旧稿”转为“完成设色的手绘彩图”。
- 颜色必须明显增强，肉眼可见植物绿、木构褐、水面青、石头灰青 / 灰褐、花草淡粉等分层。
- 保持古画、文人园林、低饱和手绘质感，避免现代游戏插画式高饱和。
- 线稿和色彩要融在一起，不能像在灰稿上简单刷了一层透明色。
- 三张图之间也要统一，不能一张偏绿、一张偏黄、一张偏灰。

## 实施路线

### 第一阶段：强风格编辑预览

1. 使用当前第一章正式图作为构图和内容基础。
2. 以第二章背景作为风格参考，进行强风格编辑：
   - 建筑木结构转为浅赭、灰褐、淡墨褐；
   - 芭蕉、竹、树丛、荷叶转为低饱和灰绿 / 青绿；
   - 水面与远景加入浅灰青层次；
   - 石头、太湖石、岸边加入灰青、灰褐、浅赭色；
   - 花草可加入极淡粉、淡红或浅紫灰，但只做点缀。
3. 保留原图的主要构图、主体位置、视角、场景空间关系和谜题线索关系。
4. 输出预览文件到 `TestPic/`，文件名标明 chapter1 场景与版本号。
5. 制作一张 chapter1 新预览与 chapter2 参考图的对照图，供确认。

### 第二阶段：必要时整张重绘

如果第一阶段结果仍然偏灰、偏旧稿，或出现补色生硬、风格不统一的问题，则进入保构图重绘：

1. 以第一章原图作为构图、视角、主体元素和线索位置参考。
2. 以第二章当前图作为画风、设色、纸底和完成度参考。
3. 按原场景重新绘制三张背景，允许笔触、树影、远景细节自然变化。
4. 重绘时必须保留兰雪堂、缀云峰、芙蓉榭各自的辨识度和玩法所需视觉锚点。
5. 新图先放入 `TestPic/` 做预览，不直接替换正式资源。

## 边界

- 不修改代码、剧情、谜题逻辑、交互流程。
- 不改 `chapter1-lanxuetang-plaque.png`。
- 第一阶段不改变构图、裁切、尺寸、文件名和代码引用。
- 第一阶段不直接覆盖正式图片。
- 若进入第二阶段重绘，允许细节和笔触变化，但主要构图、视角、主体元素、空间关系和线索位置必须保持。
- 所有正式替换都必须在预览确认后进行。

## 验收标准

- 第一章三张图整体接近第二章的手绘彩图完成度。
- 颜色明显增强，不再像淡墨灰稿。
- 木构、植被、石头、水面、远景的设色逻辑与第二章一致。
- 画面仍保持古画与文人园林气质，不出现过度鲜艳、厚涂、现代插画感。
- 构图和玩法线索没有被误改。
- 预览对照图中，第一章与第二章放在一起时不再显得像两套画风。
