# 第三章背景图 v4 按标注返工计划

状态：已完成，已替换正式资源

## 一、返工原因

v3 图虽然统一了第三章整体风格，但仍未满足最新审查意见：

- 南厅、北厅空间关系仍然过于相似，没有形成明确的南北反向关系。
- 南厅右上角门洞与上方梁架不在同一空间平面，透视关系错误。
- 南厅、北厅、留听阁的挂落、格子、家具雕花仍过多，画面偏繁。
- 留听阁未揭与剥落两张图右侧桌子仍过大，抢墙面主体。
- 草图特写线条偏规整，需要更潦草、更杂乱、更像王蘅留下的底稿。
- 所有图线条仍偏硬朗，需要更明显地软化为手绘淡墨线。

## 二、本轮硬要求

### 2.1 鸳鸯馆南厅

- 保持“秩序、整齐、维护过”的叙事气质。
- 修正右上角门洞与上方结构的空间关系，让上下处在同一个透视平面。
- 明确与北厅形成反向空间：南厅与北厅不能只是同构换摆设。
- 大幅减少复杂挂落、格子装饰和家具雕花，只保留基础梁柱、墙面、窗门、桌椅。
- 家具以平面、简洁、低存在感为主。

### 2.2 鸳鸯馆北厅

- 与南厅形成朝向相反的空间关系。
- 保留散落失败草图和潮湿动摇感。
- 减少格子栏杆、挂落、门窗密纹、家具复杂凹凸。
- 地面和家具不要过写实反光。
- 草图纸张仍需体现水面过重、桥线过弯、亭阁被压低。

### 2.3 留听阁未揭与剥落

- 两张图继续保持同一空间、同一构图、同一家具、同一光线。
- 右侧桌子必须缩小、后退、降低存在感，不能抢中央封墙。
- 去掉复杂窗格和角部装饰，只保留最简单的基本结构。
- 中央墙面继续是主体，未揭态不得泄露完整草图。
- 剥落态只改变墙面区域。

### 2.4 草图特写

- 更潦草、更杂乱、更不规整。
- 线条要像反复试探、擦改、断续留下的墙内底稿。
- 保留低位视角关系：水面、桥弧、远处建筑、竹影、倒影仍可辨。
- 不要边框、题名、印章、正式册页感。

### 2.5 全局线条与风格

- 所有图线条都要比 v3 更软，减少硬直建筑线。
- 画面保留纸本淡彩和旧纸质感。
- 不做照片、真实室内渲染、电影光影、复杂细密格子、硬黑描边。

## 三、执行方式

1. 生成 v4 候选图，过程稿保存到 `TestPic/chapter3-redraw/v4/`。
2. 五张图完成后统一尺寸到 2752×1536。
3. 对 v4 候选做更强的线条软化、暗线压浅和纸本淡彩后处理。
4. 生成 v4 对照板，人工审查是否满足本轮标注。
5. 备份当前 v3 正式图。
6. 替换正式第三章五张背景图。
7. 执行 `npm run build`，不做浏览器检查。

## 四、验收标准

- 南厅、北厅空间朝向一眼能看出反向关系。
- 南厅右上角门洞透视不再错位。
- 复杂挂落、格子和雕花明显减少。
- 留听阁右侧桌子明显缩小，不再抢墙面。
- 草图特写比 v3 更潦草、更杂乱。
- 所有图线条明显更软，更像手绘淡墨。
- 正式代码仍只引用 `Web/public/images/`。
