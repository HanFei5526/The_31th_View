## 一、审查后修订口径

本轮重绘按新的审查意见调整：

- 热点位置不再作为构图硬约束。图片先服务叙事、构图和美术统一，后续前端热点跟随新图调整。
- 除有特殊叙事要求的图以外，统一向第一章、第二章当前正式背景图靠拢：低饱和手绘彩图、纸本淡彩、线描清晰、颜色克制、有古画质感。
- 鸳鸯馆南厅、北厅大致保留当前元素和空间构图，只做风格统一、氛围强化和局部叙事微调。
- 留听阁相关图片需要重新梳理状态层级，原则上按三张重点图处理：封墙未揭、封墙剥落露出草图、草图完整显现/特写。
- 草图不能沿用当前正式册页式画法。可参考序章扫描页的观看对象和构图关系，但第三章草图要明显更潦草、更辅助线化、更像王蘅留下的观看草稿。
- 南厅、北厅当前牌匾和文字过大、过抢眼，且比例略失调。重绘时需要缩小牌匾视觉权重，弱化文字可读性，让它们回到环境装饰层级，不能抢走场景交互和叙事焦点。

## 二、剧情与视觉核心

第三章服务的核心反转是：

玩家先在鸳鸯馆北厅看到一批失败草图，产生“她画得不好”的判断；到留听阁后，遮盖层被剥开，玩家通过低位视角理解：王蘅的价值在于她发现了一个准确的观看位置。

因此图片必须同时守住两件事：

- “拙”：草图线条犹豫、比例不稳、水面过重、桥线过弯、亭阁被压低。
- “准”：从低位观看关系上，远香堂倒影、小飞虹弧线、梧竹幽居竹影确实能同入一眼。

王蘅不能被画成成熟画家、天才女画师、可对话人物或幽灵。她在第三章依然主要以痕迹出现。

## 三、当前图片问题概括

### 3.1 风格断层

南厅、北厅偏现代室内概念图，透视、梁架、灯笼、墙面装饰线条都过于密集。和第一章、第二章目前的纸本淡彩背景放在一起，会像另一套视觉系统。

### 3.2 文字风险

南厅、北厅墙面有大量可读或半可读文字，容易出现错字、乱码和误导。第三章关键文字应由前端渗字、叙事台词、笔记本记录呈现，背景里的书画文字只需要墨迹感。

### 3.3 留听阁提前泄露

当前 `chapter3-liutingge.png` 已经像墙面破开后露出完整园景。它应该是封墙未揭状态，只能看到被重新抹过的墙、灰泥质感和隐约痕迹。

### 3.4 草图层级错误

当前 `chapter3-sketch-revealed.png` 像正式册页，有边框、印章、题名，笔法也太稳。第三章草图应低于序章扫描页的完成度：序章扫描页是文徵明笔下的正式成图扫描件；第三章墙内草图是王蘅留下的观看位置记录，应该更粗、更迟疑、更像底稿。

### 3.5 显线反馈处理方式

当前代码已有红线显示层，但没有实际线条内容。重绘时应同步准备“朱砂旧线/定位线”资产，后续由前端接入。

修订意见：如果朱砂显线阶段只是基于“未揭封墙图”浮出淡红线，且与未揭图差别不大，优先用 CSS / 前端叠加实现，不单独生成完整图片。只有在后续确认红线需要复杂手绘形态、必须和剥落草图严格对位时，再追加独立透明线条资产。

## 五、单图规格

### 5.1 鸳鸯馆南厅

目标：

- 保留当前“端正、规矩、字画齐整”的大方向。
- 延续现有厅堂、挂轴、隔扇、家具等元素，不大改空间结构。
- 统一到第一、二章背景的低饱和纸本淡彩风格。

需要调整：

- 降低天花、梁架、灯笼、墙面装饰的线条密度。
- 缩小并弱化牌匾，避免牌匾和题字成为第一视觉焦点；牌匾比例要回到室内建筑装饰的合理尺度。
- 弱化可读文字，匾额和挂轴只保留书法墨迹感，不出现大字压场。
- 光线改为雨前黄昏，整体比前两章暗一些，但保持清晰。
- 右侧通往北厅的隔扇或门洞可以略强化，引导玩家过渡到北厅。

画面气质：

- 南厅是“秩序”。它应该安静、端正、被维护过。
- 不需要恐怖感，也不需要过度破败。

### 5.2 鸳鸯馆北厅

目标：

- 大致保留当前室内构图和散落纸张的布局。
- 让北厅与南厅形成清楚对照：湿、乱、纸张满地、像有人反复画坏又丢下。
- 关键修改在纸面内容。现在像平面图，需改成潦草园林观看草图。
- 牌匾和墙面文字需要降级为环境元素，不能比散落画纸更醒目。当前牌匾字太大、比例太重，重绘时需缩小、退后、压低对比度。

草图要求：

- 5 到 7 张散落纸。
- 至少 3 张可作为交互重点：水面过重、桥线过弯、亭阁被压低。
- 纸张上能看见亭、桥、水面、竹影、倒影等元素，但比例明显不稳定。
- 线条有擦改、重画、断线、犹豫，不要像工整建筑图。
- 可以有水痕、揉皱、墨点，但不要脏到看不清。

画面气质：

- 北厅是“动摇”。玩家应自然产生疑问：如果“画非一人”指向她，为什么这些画这么不成熟？

### 5.3 留听阁未揭封墙：`chapter3-liutingge.png`

你的理解是对的：未揭版本应该没有画，只有痕迹。

目标：

- 留听阁内景，中央是一面被重新抹过的封墙。
- 墙面完整，没有破洞，没有园景，没有草图成像。
- 可以参考原图的暗室、窗光、桌案、墙体位置，但空间构图建议重新整理，让封墙成为明确主体。

封墙表现：

- 墙面颜色比周围深一层。
- 灰泥粗糙，有轻微粉化、修补边界、旧线凹凸。
- 凹凸线痕只能隐约可见，像玩家第一次摸到的“有方向、有弧度”的线。
- 不能出现可直接辨认的远香堂、小飞虹、竹影。

场景气质：

- 空、静、湿、压低。
- 右侧或侧后方可有窗格暗光，照出灰泥差异。
- 左侧可保留简朴桌案、笔筒或几张纸，作为画中工作痕迹。

### 5.4 朱砂显线效果：优先 CSS / 前端叠加

这是残砚使用后的中间反馈层。

优先方案：

- 不单独生成完整图片。
- 在留听阁未揭封墙图上，用 CSS 或前端叠加一组淡红旧朱砂线。
- 线条颜色偏旧朱砂，透明度低，带轻微渗出/浮现动画。
- 不做现代霓虹发光，不让红线看起来像扫描 UI。
- 如果用 CSS 线条难以做出手绘感，可用少量 SVG path 或 mask 辅助。

作用：

- 让“残砚先显线”有明确视觉反馈。
- 为“断簪沿线剥灰”提供操作逻辑。

备用方案：

- 如果后续实际调试发现 CSS / SVG 线条无法和剥落草图对位，或视觉过于机械，再新增透明 PNG：`chapter3-liutingge-redlines.png`。
- 该 PNG 只包含墙面区域内的淡红线，不包含室内背景。

### 5.5 剥落露出草图的场景态：`chapter3-liutingge-revealed.png`

你的理解也对：这张图表现遮盖层已经被剥落，墙内草图露出来。

建议新增这一张，原因是当前只有未揭背景和一个居中草图面板，缺少“墙面从封住到露出”的空间连续性。新增后，灰泥剥落的结果会更像发生在留听阁墙上。

目标：

- 仍然是留听阁内景。
- 中央封墙的一部分灰泥已经剥落，露出墙内草图。
- 剥落边缘粗糙，有灰泥碎裂、旧墙底色、朱砂线和墨线。
- 草图嵌在墙上，不能像挂上去的一张正式画。

与未揭图的关系：

- 构图尽量同构，方便前端切换或淡入。
- 光线、桌案、窗格位置保持连续。
- 变化集中在封墙区域。

### 5.6 草图完整显现/特写：`chapter3-sketch-revealed.png`

这张需要完全重绘，当前版本不适合。

参考关系：

- 可以看序章 `scan-painting.png`，理解正式成图保存的观看关系。
- 第三章草图要比序章扫描页潦草很多。序章扫描页是文徵明完成后的正式图像扫描页；第三章草图是王蘅留下的低位视角底稿。

草图内容：

- 必须包含远香堂倒影、小飞虹弧线、梧竹幽居竹影这三个观看要素。
- 构图从低处出发，水面占比偏大，桥线弯得明显，亭阁被压得偏低，竹影靠近画面边缘。
- 可以出现辅助线、定位线、擦改痕、重复试线。
- 线条粗细不稳，部分结构画错后又重新盖线。
- 墨色可以不均匀，有断笔、洇墨、干笔。

禁止方向：

- 不要边框、题名、印章、正式画轴装饰。
- 不要成熟工笔、完整设色、平稳成稿感。
- 不要画成序章扫描页的低配复制。它应更像“视角记录”，完成度低，但空间关系准确。
- 不要在图里生成清晰题字正文。题字正文继续交给前端渗字 UI。

使用方式建议：

- 如果前端保留当前居中面板做法，这张可以作为草图特写资产。
- 如果前端改为直接显示 `chapter3-liutingge-revealed.png`，这张可用于拓片、笔记本或玩家聚焦查看。
- 两种方式可在出图后根据实际效果决定。

## 十一、返工补充原则

第一次替换后的图片偏写实，已经偏离第一章、第二章形成的视觉基准。后续返工必须遵守以下最高原则：

1. 风格必须回到第一章、第二章的背景图体系：纸本淡彩、低饱和手绘彩图、清楚线稿、轻设色、旧纸底、适度留白。可以参考 `TestPic/拙政园三十一景资料图/DP235621.jpg` 等资料图的古画线稿气质，但不能做成照片式、写实暗室、厚重光影或电影概念图。
2. 南厅和北厅本质上是同一建筑中的南北两个空间。它们的梁柱、门窗、墙面、透视、纸本质感和色彩语言必须统一，只能通过整齐/杂乱、光线强弱、地面画纸等叙事元素区分，不能像两个不同项目里的场景。
3. 留听阁未揭版本和剥落版本必须是同一个空间、同一个构图、同一套家具和光线。两张图的差异只能集中在中央封墙区域：未揭图是完整封墙和隐约痕迹，剥落图是在同一墙面上灰泥脱落、露出草图。其他区域不能变化。
