# 第二章背景图风格重绘计划

状态：已完成

## 目标

参考 `Web/public/images/common/landing_bg.png` 的古画册页风格，对第二章画中世界相关背景图进行统一重绘，使第二章视觉更接近纸本淡墨、低饱和、线描皴擦的整体气质，并保持现有剧情、谜题、热点位置和场景节奏稳定。

本轮重绘以“风格统一”为主，不大幅改动原图的色调关系、主体元素和整体构图。画面应保留第二章现有的温润、清透和可读性，避免把场景处理得过灰、过白、雾气过重，或出现过强、过密的纸纹和线条质感。

## 参考风格提炼

- 纸本底色：温润旧纸，轻微颗粒、折痕和泛黄边缘。
- 线条语言：以灰黑细线、干笔、淡墨层次为主，建筑与山石保留明确轮廓。
- 色彩控制：低饱和，主色为米黄、墨灰、淡赭、浅青灰；少量朱红只用于印章、木构或线索强调。
- 空间处理：远景留白充分，山水和园林以散点透视组织，不追求现代写实透视。
- 画面密度：主要可交互区域保留识别度，周边以水墨虚化和留白承接 UI。
- 克制处理：纸纹、皴线、墨点只作为轻度风格统一，不能形成厚重纹理或密集线网。
- 明度控制：保持温暖纸色和清晰层次，禁止整体发灰、发白、雾蒙蒙或低对比到影响辨识。

## 单图要求

### 远香堂

- 构图保留室内向外看的空间关系，画案在右下或中下区域，墙面诗轴和匾额在左侧。
- 尽量延续现有远香堂的元素布局和视角，不重新设计为完全不同的室内结构。
- 匾额和诗轴上的文字只需呈现为古风墨迹，不依赖图片中文字可读性；实际谜题文字仍由前端交互承载。
- 窗外湖面、远山、荷叶保持淡而空，避免抢夺诗词比对 UI 的注意力。
- 画案、墨迹、砚台位置要清楚，方便玩家理解“刚研过墨迹”的叙事提示。
- 室内梁柱线条应清楚但不过黑不过密，不能压住画面呼吸感。

### 小飞虹

- 桥体为核心识别对象，保持横向廊桥和弯曲桥身的视觉中心。
- 尽量保持现有小飞虹的桥、水面、荷塘、岸边植物和远景分布，不改成另一张构图。
- 左下桥脚石台附近保留可点击线索区域，画面不能过暗或过密。
- 荷塘以淡墨与少量浅粉点染处理，水面留白更多。
- 可保留一位远处人物剪影，但人物不能成为视觉主角。
- 水面和天空不做大面积灰雾，保留温暖底色和明确的桥体轮廓。

### 章节过场

- 横卷感更强，适合在转场中铺满宽屏。
- 远香堂、小飞虹、荷塘、远山可作为中园意象散布，但不需要写实对应单一地点。
- 左侧区域需有稳定视觉重心，因为现有转场样式使用 `background-position: left center`。
- 保持现有过场的横向展开和疏朗节奏，不增加过多山石、树木或墨线细节。

## 风险与控制

- 热点偏移风险：小飞虹的砚台热点当前在场景坐标约 `22%, 78%`，重绘时必须保留对应区域的桥脚石台或明显地面落点。
- UI 遮挡风险：底部叙事框、右侧笔记本会遮挡画面，主体内容应避开底部中央和右侧边缘过密区域。
- 文字误生成风险：图片里的匾额、题诗不作为谜题文本来源，生成时要求“墨迹感、不可清晰逐字阅读”，避免错字影响玩家判断。
- 风格漂移风险：统一限定为纸本淡墨、细线、淡赭灰，不使用现代厚涂、强透视、鲜艳水彩或电影感光影。
- 过度仿旧风险：不使用强纸纹、大片灰雾、泛白蒙版、脏污噪点或过密皴擦来制造“古意”。
- 构图漂移风险：生成提示词必须明确“参考原图构图与主体布局”，避免生成出看似好看但不能承接现有交互的全新场景。

## 验收标准

- 三张图在整体气质上与 `landing_bg.png` 接近：纸本、淡墨、低饱和、线描清晰。
- 三张图与原第二章图片的主体元素和大构图基本一致，只做风格重绘，不做叙事空间重设。
- 画面保持温润清晰，不发灰发白，不雾气蒙蒙，纹理和线条不过强不过密。
- 场景功能不变：远香堂仍能看出室内、诗轴、画案；小飞虹仍能看出桥、荷塘、桥脚线索区域；过场仍适合章节转入。
- 在桌面宽屏和移动窄屏下主体不被严重裁掉，底部叙事框与右侧笔记本不会遮住关键线索。
- 确认前不覆盖旧图；确认后使用新图覆盖原正式文件名，代码仍引用正式路径。

## 审批后执行顺序

1. 生成三张草案并放入 `TestPic/chapter2-redraw/`。
2. 逐张检查风格、构图、线索区域和 UI 遮挡。

4. 输出最终版到 `Web/public/images/chapter2/`，先保留版本文件名供确认。
5. 确认后覆盖原正式图片，并让第二章和过场继续引用原正式路径。
6. 启动前端检查第二章远香堂、小飞虹、章节过场的实际显示效果。
7. 通过后将本计划移动到 `当前改动计划/已通过/`。
