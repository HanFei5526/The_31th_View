# 序章古画画幅对齐与研讨按钮下移

## 修改目标

根据当前截图反馈，序章古画画幅需要略微放大，并让左右边界尽量与底部叙事对话框平齐；“开启推理研讨”按钮需要再向下移动一点。

## 涉及文件

- `Web/src/components/painting-viewer.js`
- `Web/src/styles/hud-bar.css`

## 主要改法

- 古画查看器尺寸优先按叙事对话框宽度比例（约 `55vw`）计算外框宽度。
- 保留右侧 AI 笔记本避让逻辑，避免空间不足时再次重叠。
- 将 `.prologue-synthesis-btn` 的响应式 bottom 值整体下移。

## 完成标准

- 画幅左右边界在常规宽屏下接近底部对话框左右边界。
- 画幅比上一版略大。
- “开启推理研讨”按钮比上一版更靠下，但不压住底部叙事框。
- `npm run build` 通过。
