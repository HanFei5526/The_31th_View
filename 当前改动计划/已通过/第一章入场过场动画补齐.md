# 第一章入场过场动画补齐

## 修改目标

序章完成后进入第一章时，补上与现有序章/菜单章节过场基本一致的全屏入场动画，避免玩家直接落入兰雪堂场景。

## 涉及文件

- `Web/src/components/intro-transition.js`：新增复用的章节入场过场工具。
- `Web/src/pages/menu.js`：改为调用复用工具，保持菜单进入章节的现有表现。
- `Web/src/pages/prologue.js`：序章跌入画中转场结束后，先播放第一章入场过场，再切换到第一章。

## 主要改法

- 复用现有 `.intro-transition-overlay`、`.prologue-transition-layout`、`.intro-prologue-title`、`.intro-prologue-text` 样式。
- 第一章文案优先使用现有菜单中的“第一章 · 东园 / 兰雪堂至芙蓉榭 / 水面倒影里……”。
- 背景继续使用现有 `/images/过渡页.png`，后续可在新工具配置中替换。
- 保留空格 / Z 键快进能力，并清理计时器与键盘监听。

## 完成标准

- 从菜单选择第一章时仍显示原有样式的过场。
- 序章完成后进入第一章前显示同款第一章过场。
- 过场结束后进入第一章兰雪堂，原第一章流程不被改变。
- `npm run build` 通过。
