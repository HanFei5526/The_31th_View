# 图片资源清理与章节归档方案

## 目标

- 删除 `Web/public/images/` 中当前前端完全没有引用的图片。
- 将仍在使用的图片从根目录平铺改为按章节/通用资源归档。
- 同步更新前端代码与样式中的图片路径，保证运行行为不变。

## 删除清单

以下图片在 `Web/` 源码中未找到引用，作为本次删除对象：

- `Web/public/images/过渡页.png`
- `Web/public/images/prologue-bg-4.png`
- `Web/public/images/transition-bg-0.png`
- `Web/public/images/transition-finale.png`
- `Web/public/images/transition-landing.png`

## 目录方案

- `Web/public/images/common/`
  - 菜单、着陆页、通用卷轴 UI、角色立绘。
- `Web/public/images/prologue/`
  - 序章工作室、扫描件、序章/终章共用的进入过渡底图。
- `Web/public/images/chapter1/`
  - 第一章场景图与兰雪堂匾额。
- `Web/public/images/chapter2/`
  - 第二章场景图、残砚正反面。
- `Web/public/images/chapter3/`
  - 第三章场景图、草图显影、第三章过渡图。
- `Web/public/images/finale/`
  - 终章真相空间、第三十一景线稿/低视角/完成图、结局三花园图。

## 需更新引用

- `Web/src/pages/*.js`
- `Web/src/components/*.js`
- `Web/src/styles/*.css`

## 验收标准

- `npm run build` 通过。
- `Web/public/images/` 中剩余图片均能在前端源码或样式中找到引用。
- 工作区变更只包含本次图片清理、路径更新和本计划文档。
