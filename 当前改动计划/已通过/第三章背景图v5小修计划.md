# 第三章背景图 v5 小修计划

状态：已完成，已替换正式资源

## 一、修正目标

v4 整体方向已通过，本轮只处理剩余小问题：

- 南厅门应像北厅一样开在侧边墙上，让正面墙面积更完整、更大。
- 南厅、北厅顶部残留挂落继续删减，只保留基础梁柱结构。
- 南厅、北厅、留听阁未揭、留听阁剥落四张图的视角尽量统一，避免北厅进深明显大于南厅。
- 留听阁剥落图必须使用最终潦草草图内容，不再使用墙上较完整的草图。
- 留听阁未揭与剥落图尽量使用同一底图，保证视角、家具、窗、墙体一致。

## 二、执行方式

1. 南厅 v5：重新生成小修版，保持左侧水景，右侧门改到侧墙，正面墙扩大，顶部无挂落。
2. 北厅 v5：重新生成小修版，保持右侧水景，左侧侧墙门，视角与南厅接近，减少进深和顶部挂落。
3. 留听阁剥落 v5：以当前未揭图为底，局部合成最终潦草草图到中央墙面，保证与未揭图完全同构。
4. 保持 `chapter3-sketch-revealed.png` 使用最终潦草草图。
5. 保存过程稿到 `TestPic/chapter3-redraw/v5/`，正式替换前备份当前 v4。
6. 替换正式资源后执行 `npm run build`，不做浏览器检查。

## 三、验收标准

- 南厅正面墙完整，门在侧墙，不挤占正墙。
- 南厅、北厅顶部没有明显小挂落残留。
- 南厅、北厅视角和进深接近。
- 留听阁未揭与剥落图除墙面草图外完全一致。
- 剥落墙面使用潦草草图，而不是完整工整图。
