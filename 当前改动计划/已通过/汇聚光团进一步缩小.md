# 汇聚光团进一步缩小

## 修改目标

古画中央汇聚提示需要进一步缩小，最大视觉效果接近当前截图标出的大小，并去掉外圈环带，只保留简单光团。

## 涉及文件

- `Web/src/styles/painting-viewer.css`

## 主要改法

- 将 `.pv-convergence-point` 尺寸从 58px 缩小到 34px。
- 将脉冲动画从缩放扩散改为轻微亮度/透明度呼吸。
- 使用单一柔和径向光团渐变，不再形成明显环带。
- 点击态保持金色淡出，不再扩散成大圈。

## 完成标准

- 汇聚光团最大视觉尺寸明显小于当前版本。
- 光团外侧没有明显环带。
- `npm run build` 通过。
