# 第一章图片引用对齐现有资源

## 修改目标

第一章画中世界当前引用的三张图片文件名与 `Web/public/images/` 中实际存在的资源不一致，需要改回现有资源名，避免兰雪堂、缀云峰、芙蓉榭背景加载失败。

## 涉及文件

- `Web/src/pages/chapter1-paint.js`

## 主要改法

- 将兰雪堂背景引用改为 `/images/chapter1-lanxuetang.png`。
- 将缀云峰背景引用改为 `/images/chapter1-zhuiyunfeng.png`。
- 将芙蓉榭背景引用改为 `/images/chapter1-furongxie.png`。
- 不改动第一章交互、热点、谜题逻辑。

## 完成标准

- 第一章代码引用的三张背景图均存在于 `Web/public/images/`。
- `npm run build` 通过。
