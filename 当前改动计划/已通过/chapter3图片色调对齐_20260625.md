# 第三章图片色调对齐计划

## 目标

将第三章正式场景图的色调向第一章、第二章现有正式图靠拢：提亮画面、削减灰度和雾蒙蒙感，并按物体元素分别调整颜色，避免整张图统一覆盖颜色。

## 调整原则

1. 木构、墙面、纸面、植被、水面、朱砂线分别按颜色区间处理，不做全图单一滤镜。
2. 优先对齐第一章、第二章已经形成的颜色语言；不额外保留所谓“西园章节气质”，避免用偏暗、偏旧、偏灰来制造差异。
3. 留听阁显线版需要重点提亮，但朱砂线仍要清晰可读。
4. 草图展示版允许保留一定脏污和墙面旧痕，只做轻微提亮、去灰、增强墨线可见度，不追求干净。
5. 留听阁显线/显现版是重点清理对象，需要显著减少墙面脏污和大片斑驳，但朱砂线与草图仍要清晰可读。
6. 修改正式图前先备份原图到 `TestPic/chapter3-tone-backup-*`。
7. 生成调色前后对照图，便于检查。

## 追加调整：降低做旧和污渍感

用户反馈第三章图仍有较明显的做旧斑驳、污渍和纸面痕迹，而第一章、第二章参考图整体更干净。追加处理原则如下：

1. 墙面、纸面、灰泥、地面优先减少斑驳和污渍。
2. 保留草图线、朱砂线、建筑边线和水墨轮廓，不做整图模糊。
3. 第三章不需要额外压旧、压暗；污渍纹理不能成为视觉主导。
4. 生成包含第一章、第二章参考图与第三章处理后图的总对比图。

## 追加调整：按室内/室外参照分流

用户进一步明确：第三章多数为室内场景，室内以 `chapter2-yuanxiangtang.png` 为主要参照；室外园景以 `chapter2-xiaofeihong.png` 为主要参照。追加处理原则如下：

1. 室内墙面、纸面、地面向远香堂的明亮米白靠拢。
2. 室内木构向远香堂的低饱和灰褐靠拢，压低橙色旧斑。
3. 窗外水面、植物按小飞虹的淡青灰和低饱和青绿处理。
4. 亮度提高，对比度增强，饱和度降低，整体更淡雅。
5. 不做整图统一滤镜，继续按墙面、木构、植物、水面、朱砂线、墨线分区处理。

## 追加调整：回收发白与高对比

用户反馈上一版发白、对比度偏高，仍能看到明显做旧痕迹。追加处理原则如下：

1. 室内继续以远香堂为主参照，但恢复参考图中的淡黄纸色，不追求白墙。
2. 室外继续以小飞虹为主参照，保留温润淡黄底色。
3. 降低过硬对比，线条清晰但不锐利发黑。
4. 旧斑和污渍继续减少，但保留均匀纸纹和水墨层次。
5. 按墙面/纸面、木构、地面、窗外水面、植物、朱砂/墨线分别重新上色。

## 本次综合对照结论

第一章、第二章参考图的共同特点不是单纯泛黄，而是“暖米纸底 + 浅褐灰木构 + 灰绿植物 + 青灰水面 + 柔和墨线”。旧画感主要来自纸纹和淡彩层次，不依赖大片污渍、暗角或灰雾。

第三章当前正式图的主要问题如下：

1. 墙面、地面和纸面灰雾偏重，尤其南厅、北厅、留听阁未揭版，整体像盖了一层灰色罩色。
2. 留听阁两张图的墙面剥落、污渍和做旧纹理面积过大，视觉上比草图和线索本身更抢眼。
3. 木梁、柱子和家具偏深褐，部分已有调色稿又转向橙褐，需要回到第二章远香堂那种低饱和灰褐。
4. 窗外水面和植物偏冷灰、偏混浊，应该向小飞虹和芙蓉榭的淡青灰水面、灰绿荷叶靠拢。
5. 草图显现图可以更旧，但不能“雾化到看不清”；墨线和朱砂辅助线要可辨，但不能锐化成现代标注线。

| 元素 | 主要参考 | 调整方向 |
| --- | --- | --- |

| 木梁 / 柱子 / 家具 | 第二章远香堂 | 浅茶褐、灰褐，降低橙色和黑度，保留结构线但不压暗 |

| 植物 / 荷叶 / 竹 | 小飞虹、芙蓉榭、兰雪堂 | 灰绿、淡青绿、少量深墨绿，避免统一发灰 |

| 朱砂辅助线 | 第二章残砚叙事色 | 旧朱砂、淡红褐，低透明但能被看见，不做鲜红发光 |

## 分图处理计划

1. `chapter3-yuanyang-south.png`
   - 重点：提亮室内墙面和地面，木构降橙、降黑，窗外水面和植物补淡青灰与灰绿。
   - 目标：接近远香堂的清亮室内感，同时保留鸳鸯馆南厅的安静、整齐。

2. `chapter3-yuanyang-north.png`
   - 重点：北厅可以比南厅稍乱稍冷，但亮度不能明显低一档；散落草图纸张要偏米白，地面不要灰糊。
   - 目标：用物件和秩序变化表达压抑感，不靠整体变暗。

3. `chapter3-liutingge.png`
   - 重点：整面墙提亮到暖米灰，减少四周大块斑驳和污渍；右侧桌椅降存在感但保留清晰轮廓。
   - 目标：墙面仍旧，但干净、可等待揭示，不提前抢戏。

4. `chapter3-liutingge-revealed.png`
   - 重点：必须保持原图元素、构图、结构、墙面剥落范围、窗外景、右侧桌子和草图位置一致。只减少污迹、灰雾、脏色罩层和过重污斑，不把墙皮剥落痕迹整体抹掉。草图墨线略增强，朱砂线保留淡红褐。
   - 目标：玩家第一眼看到“墙里有草图”，同时仍能看出墙皮剥落露出痕迹，而不是变成一面全新的干净墙。

5. `chapter3-sketch-revealed.png`
   - 重点：整体轻微去灰提亮，墨线增强可读性，保留草稿的犹豫、粗糙和必要旧痕。
   - 目标：像墙面局部被记录下来的草图证据，允许旧和脏存在，但不能灰到看不清。

6. `chapter3-transition-bg.png`
   - 重点：转场大图整体向第一、二章纸本淡彩靠拢，降低黑线和灰山的压迫，补一点暖纸底。
   - 目标：作为章节过场时不显得突然黑白化，也不抢过正式场景图。
