# 第一章：兰雪堂静态背景转动态（竹子摇曳）方案

## 1. 目标
将第一章场景1（兰雪堂）的单层静态背景图（`chapter1-lanxuetang.png`）升级为带有局部动态效果的背景，具体表现为“竹子随风左右摇曳”。

## 2. 核心技术挑战
目前 `chapter1-lanxuetang.png` 是一张平铺的**单层静态图片**，竹子并没有独立图层。如果直接对整张背景图使用 CSS 动画（如缩放、旋转或倾斜），会导致画面中的建筑、地面等静态景物跟着一起扭曲变形，显得极不自然。

## 3. 拟选方案设计

为了在不修改原图（不抠图切分图层）的前提下实现“仅竹子摇曳”的错觉，我建议采用**「局部遮罩分层 + 变形动画」**的技术方案：

### 方案细则：CSS Mask + 局部形变 (推荐)
1. **图层克隆**：在 `chapter1-paint.js` 中，保留原本的 `_lanxueEl` 作为静止底层。在它之上叠加一个完全重合的新图层（`_bambooLayer`），背景图依然是 `chapter1-lanxuetang.png`。
2. **局部遮罩 (Mask-Image)**：利用 CSS 的 `mask-image: radial-gradient(...)` 或 `linear-gradient(...)`，将 `_bambooLayer` 裁剪得**只露出竹子所在的区域**，其余部分透明。
3. **风动动画 (Keyframes)**：
   - 为该遮罩图层添加 CSS 关键帧动画 `@keyframes sway`。
   - 设定 `transform-origin: bottom center;`（以竹子根部或画面偏下位置为轴心）。
   - 使用微小的 `transform: skewX(...) rotate(...)` 在 `-1deg` 到 `1deg` 之间缓慢往复摇曳，模拟风吹。
4. **视觉融合**：由于上下两层图片本身完全一致，被遮罩裁剪出的竹子图层在摇曳时，只会露出底层的竹子边缘（会有一点错位模糊，但正好契合水墨画中风吹树叶的“动态虚影”效果）。

*注：还有一种使用 SVG `<feTurbulence>` + `<feDisplacementMap>` 的水波纹扭曲方案，动态更细腻，但在性能开销上较大且容易让水墨线条破碎，暂作为备选。*
