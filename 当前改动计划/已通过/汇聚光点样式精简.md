# 汇聚光点样式精简

## 修改目标

综合研讨通过后，古画中央的汇聚光点需要更小、更克制；去掉旁边提示小字和明显的白色光带。

## 涉及文件

- `Web/src/components/painting-viewer.js`
- `Web/src/styles/painting-viewer.css`

## 主要改法

- 删除汇聚点旁边的 `点击查看` 文案节点。
- 将汇聚点点击热区和可见光圈从 96px 缩小。
- 调整光圈渐变，去掉偏白外带，只保留柔和金色光晕。
- 调整点击爆开态，不再出现白色扩散带。

## 完成标准

- 古画中央汇聚点无提示小字。
- 中央光圈比当前版本更小。
- 光圈外围不再出现明显白带。
- `npm run build` 通过。
